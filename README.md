# iih - irc client

## Dependencies

* [`ii`](https://tools.suckless.org/ii/) - from suckless.org

## Install

```bash
$ npm install --global my-ink-cli
```

## CLI

```
$ iih --help

  Usage
    $ iih --host $host --channel $channel

  Options
    --host your.host
    --channel '#channel'

  Examples
    $ iih --host irc.libera.chat --channel '##spam'
```
